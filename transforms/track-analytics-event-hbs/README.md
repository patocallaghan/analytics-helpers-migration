```

yarn global add analytics-helpers-migration
analytics-helpers-migration track-analytics-event-hbs path/of/files/ or/some**/*glob.hbs
```

```
node ./bin/cli.js track-analytics-event-hbs path/of/files/ or/some**/*glob.hbs
```
